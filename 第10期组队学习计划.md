# Datawhale 组队学习

第10期Datawhale组队学习计划马上就要开始啦！

这次学习动手学深度学习PyTorch版，采用线上视频学习，最后需要学员组队完成课程作业任务。

大家可以根据我们的开源内容进行自学，也可以加入我们的组队学习一起来学。

**Part1 2月11日 14：00 -15：30**

- 开营介绍&组队（1天）

# 具体规则
- 注册 CSDN 或 Github 账户。
- 按照任务安排进行学习，完成后写学习笔记Blog。
